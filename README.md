功能描述：  
-
有一个数据文件，保存了100天 2000 个教职工/学生的所有防疫信息，请设计一个命令行程序， 支持查询某人在某一天的疫情情况，  查询某种数据的周/月的疫情统计情况，并用柱状图显示统计结果。（由于各种原因，我的实际测试数据较少）

开发环境：  
-
本项目使用Windows10+eclipse+jdk1.8.0
项目结构简介：
-
本项目主要功能是疫情上报，主要分为两个模块，有用户模块和管理员模块，用户和管理员有不同的权限，分别执行不同的功能
